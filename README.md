# 🚀 Doctor UI - MediGhar

## 👨‍⚕️ Overview
This project includes a fully functional Doctor Dashboard built using React and Tailwind CSS.

## 🔥 Features
- Responsive Sidebar Navigation
- Doctor Dashboard Overview
- Appointments Management
- Availability Setup
- Earnings Tracking
- Patient Management

## 🧠 Tech Stack
- React.js
- Tailwind CSS
- React Router
- Axios

## 📂 Routes
- /doctor → Dashboard
- /doctor/appointments
- /doctor/availability
- /doctor/earnings
- /doctor/patients

## 💡 Highlight
Developed a modular Doctor UI feature inside a scalable healthcare application.
